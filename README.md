# messengers_aggregator
Messenger Aggregator — мобильное Android-приложение, предназначенное для агрегации входящих сообщений (уведомлений) из популярных мессенджеров:
* Telegram
* WhatsApp
* Viber

Приложение перехватывает системные уведомления, отображает их в единой ленте с указанием источника и отправителя, а также позволяет отвечать на сообщения через стандартные механизмы Android (Direct Reply / Intents).

Проект ориентирован на практичную реализацию без вмешательства в работу оригинальных мессенджеров и без нарушения их внутренних протоколов.

## Ограничения и особенности
* Приложение не является альтернативным клиентом мессенджеров
* Отправка новых сообщений возможна только:
  - через Direct Reply (если поддерживается)
  - ерез системные Intents
* WhatsApp и Viber не предоставляют публичных API для сторонних клиентов
* Telegram может быть расширен в будущем через отдельную интеграцию (опционально)
